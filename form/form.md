# Forms

Manage your PDF forms.

* List form fields
* Remove fields by id
* Lock fields by id
* Unlock fields by id
* Reset fields by id
* Export form to JSON
* Fill form via JSON
* Multifill forms via JSON/CSV

The focus of this command set is manipulating forms that were created using `pdfcpu create`.
